# HW #1 Report

## 구현 방법
기존 수업에서 사용하였던 코드를 활용해서 과제에 맞게끔 개조를 해서 구현을 했습니다.

## 어려웠던 점
CRect와 CCircle에 대한 클래스를 별도로 만든 다음 배열 형태로 선언을 했지만, 복수개의 도형을 출력하는데에 실패했습니다.
MFC 자체가 C++ 기반이긴 하지만, 윈도우에서 별도로 개조해서 만든 언어이기 때문에 그런지 기존 C++과 다른 부분이 많아 그에 따른 괴리감이 느껴져 과제를 진행하는데에 전반적으로 어려움이 있었습니다.

## 기타 문제점
솔루션을 'Release'가 아닌 'Debug'로 설정하면 문제가 발생합니다. 수업시간에도 질문을 했고, 이후에 수정을 하려 했으나, 어디가 문제인지 발견하지 못했습니다.
