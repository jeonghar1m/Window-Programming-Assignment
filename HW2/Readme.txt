# HW #2 Report

## 구현 방법
기존 수업에서 사용하였던 코드를 활용해서 과제에 맞게끔 개조를 해서 구현을 했습니다.

## 어려웠던 점 및 문제점
1. 더블 버퍼링을 구현했으나, memDC를 사용하면 도형이 정상적으로 출력되지 않는 문제가 있으며, 우클릭으로 도형을 지울 때 잔상이 남습니다.
2. 좌클릭으로 그렸을 때, 도형의 갯수를 증가시켜 출력하는 부분은 구현하였으나, 지웠을 때 차감되는 부분은 구현하지 못했습니다.
3. 도형 변경 및 색깔 변경을 했을 때, 기존에 그려졌던 도형까지 새로운 형태로 변경되는 문제가 있으며, 해결하지 못했습니다.
